# Simple Node REST API

Just a simple Node REST API for me to practice

## Installation

Clone or download this repository:

```bash
git clone https://github.com/machadomatt/simple-node-api.git
```

Install packages:

```bash
npm install
```

## Usage

Set the server port:

```javascript
// server.js

const port = 3003
```

Start server:

```bash
npm start
```
